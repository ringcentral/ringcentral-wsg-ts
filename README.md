## Installation

```
yarn add ringcentral-wsg
```

or

```
npm install ringcentral-unified --save
```

Then you should be able to import the SDK like this:

```ts
import Wsg from 'ringcentral-wsg';
```

or

```js
const Wsg = require('ringcentral-wsg').default;
```
